# CELL

**CELL**은 HTML5 Canvas로 만든 웹 기반 탑다운 로그라이크 생존 게임입니다.

무기를 강화하는 게임이 아니라, 플레이어 자체가 변이하며 점점 통제 불가능한 생체 괴물이 되어가는 게임입니다.

## 플레이 방법

- `index.html`을 브라우저에서 열면 바로 실행됩니다.
- 키보드: `WASD` 또는 방향키로 이동
- 모바일/터치: 화면을 누른 채 드래그해서 이동
- 공격은 자동으로 발생합니다.
- 적을 처치하면 생체질량 경험치를 얻습니다.
- 레벨이 오르면 게임이 잠시 멈추고 3개의 변이 선택지가 나타납니다.

## 핵심 루프

1. 적들이 계속 생성됩니다.
2. 플레이어는 움직이며 자동 공격으로 적을 처치합니다.
3. 처치한 적에게서 생체질량을 흡수합니다.
4. 레벨업할 때마다 새로운 장기와 능력이 변이합니다.
5. 변이는 전투 방식과 외형을 동시에 바꿉니다.
6. 특정 변이 조합은 더 강한 진화 형태를 해금합니다.

## 변이 종류

### 공격 장기

- 촉수 증식
- 산성 주머니
- 감전 기관
- 산성 포자낭

### 감각 장기

- 다중 안구
- 자동 조준 강화
- 공격 속도 증가

### 방어 장기

- 재생 육질
- 외골격 형성
- 분열 세포
- 흡수구 개방

### 이동 변이

- 경련성 돌진
- 점액 흔적
- 경련 근섬유

## 진화 조합

특정 변이가 함께 쌓이면 플레이어가 새로운 형태로 진화합니다.

- 촉수 증식 + 감전 기관: **전기 촉수 군체**
- 산성 주머니 + 산성 포자낭: **독성 군락체**
- 재생 육질 + 외골격 형성: **불멸 생체질량**
- 분열 세포 + 흡수구 개방: **분열 포식군**

진화가 발생하면 화면 정지, 섬광, 진화명 표시, 체력 회복, 크기 증가, 변형 입자 효과가 함께 발생합니다.

## 전투 피드백

- 적을 때리면 흰색 타격 플래시, 피 튐, 숫자 피드백, 넉백이 발생합니다.
- 플레이어가 맞으면 화면 테두리와 전체 화면이 붉게 번지고 짧은 히트 스톱이 걸립니다.
- 강한 공격과 진화는 더 큰 화면 흔들림과 입자 폭발을 동반합니다.
- 처치된 적은 생체질량 흡수 입자를 플레이어에게 방출합니다.

## 적 종류

- 감염 세포
- 빠른 기생체
- 독성 유기체
- 분열 적
- 거대 생체질량 생명체

## 기술 특징

- 순수 HTML/CSS/JavaScript
- HTML5 Canvas 기반 절차적 렌더링
- 별도 이미지 에셋 없이 원, 곡선, 입자, 글로우로 생체 공포 분위기 구현
- 오브젝트 풀링으로 적, 투사체, 입자 재사용
- WebAudio 기반의 유기적 타격음, 피격음, 변이음, 진화음
- 데스크톱과 모바일 화면에 대응하는 반응형 Canvas

## 배경음악 교체

현재 배경음악은 아래 파일 경로를 사용합니다.

```text
assets/audio/bgm.mp3
```

이 위치에 원하는 배경음악 파일을 넣으면 게임 시작 시 자동으로 반복 재생됩니다.

기존의 절차적 배경 오실레이터는 제거했습니다. 타격음, 피격음, 변이음, 카드 선택음 같은 효과음은 계속 `game.js`에서 WebAudio로 생성됩니다.

## 파일 구조

```text
.
├── assets/
│   └── audio/
│       ├── README.md
│       └── bgm.mp3
├── index.html
├── styles.css
├── game.js
└── README.md
```

## 실행

별도 설치 과정은 필요 없습니다.

```text
index.html
```

위 파일을 브라우저에서 열면 게임을 플레이할 수 있습니다.
## 진화 조합 비주얼 정리

| 조합 | 진화 | 화면에서 보이는 특징 | 전투 역할 |
| --- | --- | --- | --- |
| 촉수 증식 + 감전 기관 | 전기 촉수 군체 | 촉수가 적을 찍고, 촉수 끝에서 번개가 주변 적에게 갈라져 뻗습니다. | 근접 자동 타격 + 연쇄 감전 |
| 산성 주머니 + 독성 포자 | 독성 군락체 | 화면 곳곳에 초록 독 장판이 랜덤으로 생기고, 커지면서 사라집니다. | 지역 장악 + 지속 피해 |
| 재생 점막 + 외골격 형성 | 불멸 생체질량 | 플레이어 중심에서 밝은 생체 파동이 바깥으로 퍼집니다. | 방어형 광역 파동 |
| 분열 세포 + 흡수구 개방 | 분열 포식군 | 회전 방향으로 많은 분열 투사체가 화면 끝까지 퍼져 나갑니다. | 대량 투사체 + 관통 압박 |

HUD 하단의 작은 칩은 현재 선택한 변이 스택을 요약합니다.

```text
TEN 촉수 / ACD 산성 / ELC 전기 / SPO 포자 / EYE 감각
SHL 외골격 / SPL 분열 / ABS 흡수 / SLM 점액 / DSH 돌진 / SPD 이동 성장
```
